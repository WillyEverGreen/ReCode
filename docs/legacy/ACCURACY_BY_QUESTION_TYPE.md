## ğŸ¯ **95%+ ACCURACY** (Pattern-Based Detection)

These questions are NOT in ground truth but will be detected by **pattern recognition**:

### **Any Sorting Problem**
- Pattern: Detects `.sort()`, `Arrays.sort()`, `sorted()`
- Complexity: **O(n log n)** time, **O(1)** or **O(n)** space
- Examples:
  - Sort employees by age
  - Sort array of objects
  - Merge sorted arrays
  - Kth largest element (with sorting)

### **Any Hash Map/Set Problem**
- Pattern: Detects `HashMap`, `Map`, `Set`, `dict`, `{}`
- Complexity: **O(n)** time, **O(n)** space
- Examples:
  - Count frequency of elements
  - Find first unique character
  - Group anagrams (not in ground truth)
  - Intersection of arrays
  - Union of arrays

### **Any Two Pointer Problem**
- Pattern: Detects `left`, `right`, `slow`, `fast`
- Complexity: **O(n)** time, **O(1)** space
- Examples:
  - Remove element
  - Squares of sorted array
  - Partition array
  - Dutch National Flag variants

### **Any Sliding Window Problem**
- Pattern: Detects window tracking, `right - left`
- Complexity: **O(n)** time, **O(k)** space
- Examples:
  - Max sum subarray of size k
  - Longest substring with k distinct chars
  - Minimum size subarray sum
  - Permutation in string

### **Any Binary Search Problem**
- Pattern: Detects `mid = (left + right) / 2`
- Complexity: **O(log n)** time, **O(1)** space
- Examples:
  - Find peak element
  - Search in 2D matrix
  - Koko eating bananas
  - Capacity to ship packages

### **Any DFS/BFS Problem**
- Pattern: Detects recursion or queue/stack
- Complexity: **O(V + E)** for graphs, **O(n)** for trees
- Examples:
  - Number of islands
  - Clone graph
  - Course schedule
  - Word ladder

### **Any DP Problem**
- Pattern: Detects `dp[]`, memoization, `@cache`
- Complexity: **O(n)**, **O(nÂ²)**, or **O(n*m)** based on dimensions
- Examples:
  - Min cost climbing stairs
  - Unique paths
  - Partition equal subset sum
  - Target sum

### **Any Backtracking Problem**
- Pattern: Detects recursion + choice + undo
- Complexity: **O(2^n)** for subsets, **O(n!)** for permutations
- Examples:
  - Subsets
  - Permutations
  - Combinations
  - N-Queens
  - Sudoku solver

### **Any Heap Problem**
- Pattern: Detects `PriorityQueue`, `heapq`, `MinHeap`
- Complexity: **O(n log n)** time, **O(n)** space
- Examples:
  - Kth largest element (with heap)
  - Top K frequent elements
  - Merge K sorted lists
  - Task scheduler

### **Any Trie Problem**
- Pattern: Detects `TrieNode`, `children[]`, `isEnd`
- Complexity: **O(m)** time (m = word length), **O(n*m)** space
- Examples:
  - Implement Trie
  - Search suggestions
  - Word search II
  - Replace words

### **Any Union-Find Problem**
- Pattern: Detects `parent[]`, `find()`, `union()`
- Complexity: **O(Î±(n))** amortized (inverse Ackermann)
- Examples:
  - Number of connected components
  - Redundant connection
  - Accounts merge
  - Most stones removed
